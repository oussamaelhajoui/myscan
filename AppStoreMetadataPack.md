## Description

`myscan` is a fast, simple local network scanner for iPhone and iPad.

Use it to discover hosts on your network and check common ports with real-time output while scanning. The app is built for practical diagnostics: quick checks, clear results, and an interface that stays responsive during long scans.

Key features:

- Local subnet host scanning
- Port scanning with live output
- Open and closed result visibility during scan
- Progress display and estimated time left
- Save scan results for later review
- Configurable scan behavior and range

Privacy-first design:

- No user account required
- No personal data collection by the developer
- No tracking
- No selling or sharing of personal data

`myscan` is intended for legitimate network administration, diagnostics, and troubleshooting on networks you own or are authorized to test.
